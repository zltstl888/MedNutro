MedNutro 官网 1.0 全球品牌中心搭建方案
1. 品牌定位与视觉规范 (Vibe & Style)
核心主张：精准定义高效营养新标准 。
三大理念：科学有解、跨境无界、生命当燃 。
视觉调性：
主色调：实验室白（#FFFFFF）搭配深邃紫（#24123E，对应小黑瓶）与科技蓝（#E6F0FF，对应小白瓶）。
质感：高精细度的分子动力学模拟、极简排版、大面积留白，体现“医药级”与“自然”的平衡 。
2. 页面叙事结构 (Sitemap)
第一屏：品牌宣言 (The Hero)
视觉：居中展示小白瓶或小黑瓶的高端 3D 渲染图，背景为流动的、透明的细胞光影动画 。
文案：
大标题：精准定义高效营养新标准 。
副标题：科学有解 · 跨境无界 · 生命当燃 。
执行建议：使用粘性导航栏（Sticky Header），包含：品牌起源、全球科技、核心方案、联系我们。
第二屏：创始人觉醒故事 (Brand Origin)
内容：描述从“每日百粒无效消耗”到“精准干预”的生命转折 。
关键金句：健康不是剂量堆砌，而是精准干预 。
视觉：使用黑白人像摄影结合动态文字排版，营造出从“至暗时刻”到“生命觉醒”的视觉过渡 。
第三屏：全球科技背书 (Global Engine)
核心内容：
瑞士基因：品牌在瑞士注册，传承百年精工标准与研发理念 。
战略合作：与霍普金斯·华盛顿生命医学实验室 (HWLL) 建立全链战略合作 。
加国生产：所有产品于加拿大制造，严格遵循全球 GMP 标准。
视觉：发光的全球协作地图，强调“跨境无界”的资源整合能力 。
第四屏：细胞级精准方案 (The Core Matrices)
板块 A：细胞平衡小白瓶 (IPE EPA)
定位：重建细胞自愈屏障 。
卖点：专利炎症阻断矩阵，EPA 纯度 ≥90%，总 Omega-3 ≥97% 。
执行建议：加入一个交互对比表，直观展示“MedNutro 临床级鱼油”与“普通鱼油”在纯度、杂质上的显著差异 。
板块 B：女士细胞焕能小黑瓶
定位：守护女性魅力与荷尔蒙平衡 。
卖点：搭载 HWLL 氧自由基捕获技术，22 种活性自由基清除剂 。
执行建议：展示其“抗氧化网络”的三维模型，体现谷胱甘肽、VC、VE 形成的立体防御 。
第五屏：价值升华 (Elite Assets)
核心理念：让健康成为您可被量化的精英资产 。
行动导向：非强制购买，而是“邀请加入精准营养之旅”。
